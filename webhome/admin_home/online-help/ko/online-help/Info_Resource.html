네비게이션 경로: Home > node-name > 엔진 컨테이너 > container-name > 리소스

개요

Engine Container의 리소스에는 JDBC 데이터 소스, JMS, JCA리소스, 트랜젝션이 있다. JDBC데이터 소스는 JEUS Manager에서 추가 가능하며 Engine Container에서는 단지 모니터링과 제어만 가능하다. JMS 리소스는 JMS Engine이 작동할 때만 사용 가능하다. JCA 리소스는 Resource Adapter Module을 Deploy하였을 경우 사용 가능하다. 트랜젝션은 Engine Container 마다 항상 존재하는 리소스이다.
