JEUS 매니저 리소스

네비게이션 경로: Home > node-name > JEUS 매니저 리소스

개요

JDBC, URL, 자바메일, MQ 에 대한 리소스 정보를 나타낸다. JEUS 데이터 소스는 어플리케이션에서 JDBC를 이용해 데이터 베이스에 연결하고 할 경우 사용하는 리소스이다. URL Sources는 URL을 JNDI name에 bind 하는데 사용된다. JavaMail는 SMTP와 같은 메일 프로토콜을 지원하는 것을 사용하여 클라이언트 어플리케이션으로부터 e-mail을 보내는데 사용한다. JEUS는 JEUS 매니저 서비스에 MQ 데이터 소스를 설정함으로써 타사 MQ 제품과 연동이 가능하다.
