네비게이션 경로: Home > node-name > 엔진 컨테이너 > container-name > 리소스 > JMS > 클라이언트

개요

현재 JMS 엔진에 접속 중인 클라이언트의 목록을 표의 형태로 볼 수 있다. 클라이언트 별로 다음 정보가 표시된다.

이름

JMS 엔진이 구분하는 클라이언트 이름. 클릭하면 해당 클라이언트의 정보를 보여주는 통계 페이지로 이동한다.

원격 주소

클라이언트의 원격 주소(Host Address:Port).

연결 시간

클라이언트가 JMS 엔진에 접속을 맺은 시간.

통계

특정 클라이언트의 정보를 보여주는 페이지이다. 개요 페이지에 표시된 것과 같은 이름, 호스트 주소, 포트, 연결 시간 정보가 나타난다.

연결

이 페이지는 해당 클라이언트가 JMS 엔진에 맺은 커넥션의 목록을 보여준다. 현재 JEUS MQ 클라이언트 라이브러리의 구현상 하나의 커넥션이 하나의 클라이언트로 표시된다. 각 열의 의미는 다음과 같다.

이름

JMS 엔진이 구분하는 커넥션 이름. 클라이언트 이름을 Prefix로 사용한다. 이 이름을 클릭하면 JMS 커넥션 리소스 페이지로 이동하여 해당 커넥션 정보를 볼 수 있다.

트랜잭션

이 커넥션으로부터 만들어진 세션들 중 트랜잭션 처리된(Transacted) 세션이 있는지의 여부. 트랜잭션 처리된 세션이 하나라도 있으면 true로 나타난다.

세션 수

이 커넥션으로부터 만들어진 세션의 개수.
