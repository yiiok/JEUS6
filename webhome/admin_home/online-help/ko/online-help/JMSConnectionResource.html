통계

이 JMS 커넥션의 기본 정보를 보여준다.

이름

JMS 엔진이 구분하는 커넥션 이름. 클라이언트 이름을 Prefix로 사용한다.

통계

이 커넥션으로부터 만들어진 세션들 중 트랜잭션 처리된(Transacted) 세션이 있는지의 여부. 트랜잭션 처리된 세션이 하나라도 있으면 true로 나타난다.

세션

이 JMS 커넥션으로부터 클라이언트가 생성한 세션의 목록이 나타난다. 세션 별로 다음과 같은 기본 정보가 표시된다.

이름

JMS 엔진 내에서 유일한 세션 이름. 커넥션 이름을 Prefix로 사용한다. 이 이름을 클릭하면 JMS 세션 리소스 페이지로 이동하여 해당 세션 정보를 볼 수 있다.

Durable Subscription 수

해당 세션으로부터 생성된 메시지 수신자가 등록한 Durable Subcription의 개수.

기한이 지난 메시지 수

해당 세션에서 기한이 지난 메시지의 개수. 서버 세션에서만 출력된다.

메시지 수

해당 세션을 통해 클라이언트와 주고 받은 메시지 개수.

처리되지 않은 메시지 수

해당 세션 내에서 아직 처리되지 않은 메시지의 개수. 서버 세션인 경우에 출력된다.

평균 메시지 대기 시간(밀리 초)

해당 세션에서 메시지가 생성되고 전달을 시작할 때까지 걸린 시간의 평균 값. 서버 세션인 경우에 출력된다.

평균 전송 시간(밀리 초)

해당 세션에서 생성된 메시지가 전달되는데 소요된 시간의 평균 값. 서버 세션에서만 출력된다.
