MBean 모니터링

개요

모니터링이 가능한 MBean의 정보를 실시간으로 갱신하여 볼 수 있는 XY 차트 제공한다.

MBean 모니터링 트리

현재 등록되어 있는 MBean의 리스트를 트리 구조로 제공한다.

MBean 모니터링 정보

선택된 MBean의 상세 정보와 실시간 모니터링을 제공한다.

MBean 기본 정보

현재 선택된 MBean 의 상세 정보를 보여준다.

노드 이름

현재 MBean 에 대한 해당 노드 이름을 나타낸다.

MBean 이름

선택된 MBean의 ObjectName 을 나타낸다.

클래스 이름

선택된 MBean의 관련 클래스 이름을 나타낸다.

설명

MBean의 description 정보를 보여준다.

MBean 속성

MBean 의 속성(attribute) 정보를 보여준다.

이름

해당 attribute 이름을 나타낸다.

액세스

각 attribute 의 액세스 정보를 R, W, RW 의 형태로 보여진다 (R:Read, W:Write, RW: Read & Write).

반환 형태

해당 attribute의 반환 형태(type)을 나타낸다.

값

해당 attribute의 값을 나타낸다.

MBean 작업

MBean 의 작업(operation) 정보를 보여준다.

이름

해당 operation 함수 이름을 나타낸다.

반환 형태

해당 operation의 반환 형태(type)을 나타낸다.

임팩트

해당 operation의 임팩트 값을 나타낸다 (함수의 제어 상태에 따라 ACTION, ACTION_INFO, INFO, UNKNOWN로 표현된다).

매개변수 갯수

해당 operation 함수가 갖는 매개변수 갯수를 나타낸다. 매개변수가 존재할 경우, 값을 직접 입력하여 MBean 을 직접 호출할 수 있다.

MBean 모니터링 통계

현재 선택된 MBean의 통계 정보와 모니터링 차트를 제공한다. 통계 정보는 다섯 가지(BoundedRangeStatistics, BoundaryStatistics, RangeStatistics, CountStatistics, TimeStatistics)로 구분되어 해당 정보를 나타낸다.

통계 종류

5가지의 통계 정보로 구성된다.

BoundedRangeStatistics

HighWaterMark, LowWaterMark, Current, UpperBound, LowerBound 의 5가지 속성을 갖는다.

BoundaryStatistics

UpperBound, LowerBound 의 2가지 속성을 갖는다.

RangeStatistics

HighWaterMark, LowWaterMark, Current 의 3가지 속성을 갖는다.

CountStatistics

Count 값을 보여준다.

TimeStatistics

min, max, total 시간과 count 값을 보여준다.

통계 세부 정보

해당 통계에 대한 상세 정보를 볼 수 있다.

이름

통계 테이블에서 선택된 필드의 이름을 보여준다.

설명

선택된 통계에 대한 설명(description)을 보여준다.

단위

선택된 통계의 단위를 보여준다.
