개요

JEUS JMX는 JMX Remote API 커넥터, HTML 어댑터, SNMP 어댑터와 같은 관리 객체를 가지고 있다. 이것은 JEUS 모니터링 정보에 액세스하는 방법으로 이 세 가지를 제공한다는 것이다. 이 페이지에서는 이 세 가지(JMX Remote API 커넥터는 Remote API를 지원하는 RMI와 Socket 기반의 JMXMP로 나눌 수 있다.) 방법을 통해 JMX Export Service를 설정할 수 있다.

어댑터

HTTP

HTML을 지원하는 JMX의 프로토콜 어댑터를 설정한다.

Html Adaptor Port

HTML 어댑터의 리스너 포트로, 이 포트로 웹 브라우저가 접속하게 된다. '-1’로 설정을 하게 되면 JMX Manager가 HTML 프로토콜을 사용하지 않음을 의미한다.

경고

다른 서비스가 사용하는 포트를 사용하면 안 된다는 것을 주의하기 바란다.

SNMP

JMX가 제공하는 SNMP 프로토콜 어댑터를 설정한다.

Snmp Adaptor Port

SNMP 어댑터의 리스너 포트

고급 선택사항
Snmp Version

SNMP 버전을 지정하며 1, 2 또는 3을 지정할 수 있으며, 기본 값은 3이다.

Snmp Max Packet Size

SNMP 패킷에 대한 최대값을 설정하며 최소 256바이트부터 설정 할 수 있다. 기본 값은 4096이다.

Snmp Security

보안을 적용시킬 것 인지를 설정한다. 기본 값은 ‘false’이며, ‘true’는 SNMP 버전 3에서만 지정이 가능 하다.

Trap Demon

트랩 데몬을 줄 별로 ip:port 형식으로 입력한다.

쓰레드 풀
Max

SNMP 어댑터에 pooling되는 객체의 최대값을 지정한다.

Min

SNMP 어댑터에 pooling되는 객체의 최소값을 지정한다.

Period

SNMP 어댑터에 pooling되는 객체를 정리하는 시간을 지정한다.

Connector

JMXMP

JMX 커넥터는 JMX Remote API에서 정의되어 있는, 리모트에서 JMX MBeanServer에 접근하기 위한 커넥터를 설정한다. JMXMP는 Socket 기반의 커넥터이고, 기본적으로는 JMXMP 커넥터를 사용한다.

Jmxmp Connector Port

다른 프로세스에서 이 엔진 컨테이너의 JMX로 접속할 때 사용하는 listen port를 지정한다. 만약 이 값이 0이거나 지정하지 않으면, JEUSMP 커넥터가 사용하는 listen port를 따로 만들지 않고 JEUS의 공통 port를 사용한다. 만약 JEUS의 JMX RemoteAPI를 사용하지 않고 다른 Runtime에서 JMXMP protocol로 접근하고자 한다면 이를 0이 아닌 다른 값으로 지정해야 한다.

RMI

JMX 커넥터는 JMX Remote API에서 정의되어 있는, 리모트에서 JMX MBeanServer에 접근하기 위한 커넥터이다. RMI 커넥터는 Remote API 기반의 커넥터이다.

Rmi Connector Port

다른 프로세스에서 이 엔진 컨테이너의 JMX를 접속할 때 사용하는 RMI 커넥터의 port를 지정한다. 만약 이 값이 0이면 랜덤하게 포트를 설정하게 된다.

Export Name

JEUS의 기본 export name 대신에 다른 export name을 사용할 때 설정한다. 이 export name은 JMXServiceURL의 URL path에 들어가게 된다.

Ref Export Name

이 커넥터를 얻을 수 있는 jndi name을, JEUS의 기본 jndi name이 아닌 다른 name으로 지정하고자 할 때 사용한다. 이 export name으로 lookup하면 JMXConnector 객체를 얻을수 있다.

Mlet

이 엔진 컨테이너의 MBeanServer에 등록할 MLet의 URL을 지정한다.
