생성

이 페이지에서는 새 스케줄러 작업을 생성할 수 있다.

목록

Enabled

이 작업을 실제로 구동시킬지 설정하는 T/F 플래그를 의미한다.

Class Name(클래스 이름)

jeus.schedule.ScheduleListener를 구현한 스케줄러 작업 클래스의 이름을 의미한다. 반드시 fully qualified name으로 적어야 한다.

Name(이름)

OPTIONAL. 이 작업에 해당하는 이름을 의미한다.

시작 시간

OPTIONAL. 이 작업을 최초로 수행할 시작 시간을 의미한다. XML dateTime type으로 yyyy-mm-ddThh:mm:ss.sss 형 식으로 입력한다. 없으면 서버 구동시에 시작된다.

종료 시간

OPTIONAL. 이 작업을 최종적으로 종료할 시간을 의미한다. 없으면 끝나지 않는다는 것을 의미한다.

간격

이 작업이 수행되는 시간 간격 또는 작업 주기를 의미한다. 밀리세컨드, 분, 시, 일 단위로 입력할 수 있다.

작업 수

OPTIONAL. 작업의 최대 수행 횟수를 의미한다. '-1'일 경우 제한이 없음을 의미한다.

설정

위 생성에서 설명한 부분들에 대한 설정을 수정할 수 있다. 스케줄러 서비스의 작업 목록에서 각각의 작업 중 하나를 선택하면 재설정 화면이 나온다.
