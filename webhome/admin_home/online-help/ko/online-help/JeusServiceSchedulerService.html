설정

이 페이지에서는 JEUS Scheduler Service를 제공하는데 필요한 쓰레드 풀을 설정하거나 새로운 스케줄러 작업을 추가할 수 있다.

쓰레드 풀

JEUS Scheduler는 사용자가 등록한 여러 개의 작업을 각각에 해당하는 스케줄에 맞춰서 수행해야 하기 때문에 쓰레드에 대한 관리가 필요하다.

Min

필드 설명 참조 바람.

Max

필드 설명 참조 바람.

Period

필드 설명 참조 바람.

작업 목록

새로 추가한 스케줄러의 리스트가 나타나는 부분이다. 여기에 나열된 작업들은 서버를 구동할 때마다 자동으로 스케줄링 된다.

클래스 이름

jeus.schedule.ScheduleListener를 구현한 스케줄러 작업 클래스의 이름을 의미한다.

이름

이 작업에 해당하는 이름을 의미한다.

시작 시간

이 작업을 최초로 수행할 시작 시간을 의미한다.

종료 시간

이 작업을 최종적으로 종료할 시간을 의미한다. 없다면 끝나지 않는다는 것을 의미한다.

간격

이 작업이 수행되는 시간 간격 또는 작업 주기를 의미한다.

작업 수

작업의 최대 수행 횟수를 의미한다. '-1'일 경우 제한이 없음을 의미한다.
