설정

기본 설정

분산 세션 서버를 사용할 때 엔진별 세션 서버(세션 라우터)의 기본 설정을 정의한다.

Engine Name

해당 분산 세션 서버의 엔진 이름 정보를 의미한다. 보통 EJB 엔진에서 사용할 경우 ejb-engine-name , 서블릿 엔진에서 사용할 경우 servlet-engine-name이 나타나며, 이는 해당 엔진 당 고유한 분산 세션 서버 이름을 의미한다.

Replication Group

해당 분산 세션 서버의 논리적인 그룹 이름을 지정해 준다. 이 지정된 이름은 다른 분산 세션 서버 설정에서 아래의 Backup Group의 설정에 필요하다. 즉, 다른 분산 세션 서버에서 해당 세션 서버를 되도록이면 백업 서버로 사용하고자 할때 Backup Group에 해당 설정값을 적어주면 된다.

Backup Group

특별히 선호하는 백업 분산 세션 그룹의 이름을 설정한다. 다른 분산 세션 서버 설정 중의 Replication Group의 값을 적어주면 된다. 이 값이 설정되었을 경우 설정 값의 그룹을 백업 분산 세션 서버로 선택하는데 최우선 조건으로 사용한다.

Location

해당 분산 세션 서버의 물리적인 Machine이름, 즉 위치(Location) 정보를 설정한다. 하나의 서버 이름일 수도 있고 여러 서버가 rack안에 구성된 경우 rack의 상징적인 이름일 수도 있다. 내부적으로 백업 분산 세션 서버를 구성할 때 여기서 설정된 Location 정보와 되도록이면 다른 Location에 존재하는 분산 세션 서버를 백업으로 선택할 것이다.

파일 스토리지

파일 스토리지 의 설정과 동일하다. 단, 여기서 설정한 파일 스토리지 설정은 상위 설정보다 우선하므로, 세션 라우터별 특별한 파일 스토리지를 설정할 수 있다.

생성
