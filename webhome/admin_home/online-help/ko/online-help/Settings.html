설정

개요

이 페이지에서는 WebAdmin의 전반적인 기능들에 대한 설정 및 각각의 노드 설정에 대한 백업 기능을 제공한다.

설정 - 화면

이 설정에서는 JEUS WebAdmin의 화면에 대해 설정을 한다. 모든 설정은 각 사용자의 쿠키에 저장된다.

설명 보기

각 요소 혹은 양식의 설명을 보여줄 것인가를 선택하는 값이다. "true"로 설정하면 설명이 보이고 "false"로 선택하면 설명이 사라진다. 설명이 사라지더라도 각 요소를 클릭하게 되면 해당 요소의 설명을 볼 수 있다.

언어

WebAdmin의 양식의 설명과 도움말의 언어를 어떤것으로 할 것인지 선택한다.

설정 - XML

이 설정에서는 XML을 처리할 때 필요한 옵션을 설정한다. 모든 설정은 사용자의 쿠키에 저장된다.

주석 유지

JEUS의 모든 XML처리는 JAXB를 이용한다. 기본적으로 JAXB는 주석을 유지 하지 않으므로 이 설정을 "false"로 하면 주석을 잃을 위험이 있다. 따라서 가능하면 주석을 유지할 것을 권장한다.

요소 재정렬

툴을 이용하여 설정 파일을 수정하지 않을 경우 요소의 순서를 틀린 위험이 있다. XML처리시에 XML 문서가 유효한지 여부를 항상 검사하므로 요소 순서가 틀리게 되면 오류가 발생한다. 따라서 이 설정을 "true"로 하게 되면 요소를 재정렬 하여 유효한 문서로 만들어 준다.

설정 - 새로 고침

새로 고침 설정은 JEUS WebAdmin의 뷰들의 갱신 주기를 설정한다. 각 갱신주기는 초단위로 입력이된다. 모든 설정은 사용자의 쿠키에 저장된다.

JEUS 클러스터 새로 고침 주기

JEUS 클러스터 페이지의 새로 고침 주기를 설정한다.

JEUS 노드 트리 새로 고침 주기

JEUS 노드 트리의 새로 고침 주기를 설정한다.

로그 분석 상태 뷰 새로 고침 주기

로그 분석 상태 뷰의 새로 고침 주기를 설정한다.

JEUS 매니저 연결 제한 시간

JEUS 매니저 연결 제한 시간을 설정한다.

JEUS 매니저 소켓 제한 시간

JEUS 매니저 소켓 제한 시간을 설정한다.

설정 관리

선택된 노드의 설정에 대해 백업, 적용, 삭제등 관리 기능을 제공한다.

대상이 되는 노드는 읽기 / 쓰기 권한이 있어야 한다.

노드

현재 관리의 대상이 되는 노드를 표시한다.

name

노드의 백업에 대한 이름을 표시한다. 마우스를 위에 올려 놓은 경우 백업 생성시 기록한 상세 정보를 확인할 수 있다.

time

백업의 생성 시간을 표시한다.

적용

백업을 대상이 되는 노드에 적용한다. 기존 노드는 Backup이란 이름으로 저장된다.

삭제

백업을 삭제 한다.

백업

현재 대상이 되는 노드를 백업한다. 이름과 상세 정보(설명)를 기록할 수 있다.
