URL 리소스

네비게이션 경로: Home > node-name > JEUS 매니저 리소스 > URL

설정

이 페이지에서는 해당 익스포트 이름의 URL을 수정 할 수 있다. URL을 수정한 후 확인 버튼을 누르면 URL 리소스 목록에서 수정된 것을 확인 할 수 있다. 하지만 부트가 된 상태에서 리소스 설정을 수정하였다면 재부팅을 해야 새로운 설정이 적용됨을 유의한다.

개요

이 페이지에서는 설정되어 있는 URL 리소스의 익스포트이름과 URL 목록을 표의 형태로 볼 수 있고 새로운 URL 리소스를 추가 할 수 있다.

이미 등록된 URL 리소스의 익스포트이름을 표에서 클릭하면 설정 페이지로 이동하여 해당 URL 리소스를 수정 할 수 있고 휴지통 아이콘()을 클릭하면 해당 URL 리소스를 제거할 수 있다. 새로운 URL 리소스를 추가하기 위해서는 익스포트 이름과 URL을 설정하고 추가 버튼을 누르면 URL 리소스 목록에서 추가된 것을 확인 할 수 있다. 하지만 부트가 된 상태에서 추가하였다면 재부팅을 해야 새로운 설정이 적용됨을 유의한다.
